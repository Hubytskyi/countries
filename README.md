# Gatsby Countries API

## 1. Prerequisites
### 1.1. Environment
* Node 16.x
* NPM 8.x

### 1.2. Dependencies
* [REST Countries API](https://restcountries.com/#api-endpoints-v2)

## 2. Installation

1. Run the following command to install dependencies:

        npm install

## 3. Running the app

1. Run development

        npm run develop

2. Run build

        npm run build

3. Clean public dir

        npm run clean
